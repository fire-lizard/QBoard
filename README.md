# Supported Board Games:

  1. Chess Variants
     * FIDE Chess
     * Capablanca Chess
     * Gothic Chess
     * Janus Chess
     * Grand Chess
  2. Shogi Variants
     * Shogi
     * Micro Shogi
     * Kyoto Shogi
     * Mini Shogi
     * Judkins Shogi
     * Whale Shogi
     * Tori Shogi
     * Euro Shogi
     * Yari Shogi
     * Heian Shogi
     * Sho Shogi
     * Wa Shogi
     * Chu Shogi
     * Dai Shogi
     * Tenjiku Shogi
     * Dai Dai Shogi
     * Maka Dai Dai Shogi
     * Ko Shogi
  3. Other Board Games
     * Xiangqi
     * Makruk
     * Shatranj

# Supported Engine Protocols:

  1. WinBoard
  2. UCI (Universal Chess Interface)
  3. USI (Universal Shogi Interface)
  4. UCCI  (Universal Chinese Chess Interface)
  5. Qianhong

# Export games to the following formats:

  1. FEN
  2. PGN
  3. WXF
  4. PSN
  5. CSA
  6. KIF
  7. KI2
  
Also could be possible to import game as FEN and continue it (depends on the engine capabilities).
